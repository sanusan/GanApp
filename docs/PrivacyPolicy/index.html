サポート時

  • サポート時に、問題解決のための端末種類、OSバージョン等をご提供いただく場合があります。

データ解析

  • アプリの利便性向上のため、匿名で個人を特定できない範囲で最新の注意を払い、アクセス解析をしております。
    例えばアプリのクラッシュ時にどんな原因でクラッシュしたかを匿名で送信して、バグの素早い修正に役立たせております。
